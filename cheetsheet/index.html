This HTML CheatSheet is For You!
  1. HTML Boilerplate 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>

    </title>
</head>
</html>
  2. Head tag: Head tag is used to contain all the head element in the html file. It contains the title, style, meta, … etc tag. 
    <head>Contains elements describing the document</head>
  3. Body tag: It is used to define the body of html document. It contains image, tables, lists, … etc. 
    <body>The content of your HTML page</body>
  4. Title tag: It is used to define the title of html document. 
    <title>tab name</title>
  5. Heading tag: It is used to define the heading of html document. 
    <h1>Heading 1 </h1>
<h2>Heading 2 </h2>
<h3>Heading 3 </h3>
<h4>Heading 4 </h4>
<h5>Heading 5 </h5>
<h6>Heading 6 </h6>
  6. Paragraph tag: It is used to define paragraph content in html document. 
    <p>Example: Computer science portal</p>
  7. Emphasis tag: It is used to renders as emphasized text. 
    <em>Example</em>
  8. Bold tag: It is used to specify bold content in html document. 
    <b>Bold word</b>
  9. Italic tag: It is used to write the content in italic format. 
    <i>Example</i>
  10. Small (text) tag: It is used to set the small font size of the content. 
    <small>Example</small>
  11. Underline tag: It is used to set the content underline. 
    <u>Example</u>
  12. Deleted text tag: It is used to represent as deleted text. It cross the text content. 
    <strike>GeeksforGeeks</strike>
  13. Anchor tag: It is used to link one page to another page. 
    <a href="link...">
GeeksforGeeks</a>
  14. List tag: It is used to list the content. 
    <li>List item 1</li>
<li>List item 2</li>
  15. Ordered List tag: It is used to list the content in a particular order. 
    <ol>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
    <li>List item 4</li>
</ol>
  16. Unordered List tag: It is used to list the content without order. 
    <ul>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
    <li>List item 4</li>
</ul>
  17. Scrolling Text tag: It is used to scroll the text or image content. 
    <marquee bgcolor="#cccccc" loop="-1"
scrollamount="2" width="100%">
Example Marquee</marquee>
  18. Center tag: It is used to set the content into the center. 
    Example
  19. Font tag: It is used to specify the font size, font color and font-family in html document. 
    <font face="Times New Roman">Example</font>
  20. Line break tag: It is used to break the line. 
    Example<br>A computer science portal
  21. Image tag: It is used to add image element in html document. 
    <img src="exp.jpg" width="40"
height="40" border="0">
  22. Link tag: It is used to link the content from external source. 
    <head>
<link rel="stylesheet" type="text/css" href="style.css">
</head>
  23. Form tag: It is used to create html form for user. 
    <form action="mailto:you@yourdomain.com ">
Name: <input name="Name" value="" size="80"><br>
Email: <input name="Email" value="" size="80"><br>
<br><center><input type="submit"></center>
</form>
  24. Dropdown option tag: It is used to select an option from a drop-down list. 
    <form method=post action="/cgibin/example.cgi">
<center> Select an option:<select>
<option>option 1</option>
<option selected>option 2</option>
